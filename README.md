# AgentFlow CLI

A command-line interface for browsing and installing workflows from the AgentFlow Marketplace.

## Installation

```bash
npm install -g agentflow-cli
```

## Usage

```bash
agentflow
```

- `agentflow install <workflow-id>` - Install a workflow
